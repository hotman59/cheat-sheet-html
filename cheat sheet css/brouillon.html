La propriété CSS font-family indique une liste de polices -ou de familles génériques de police à appliquer à l'élément sélectionné, par ordre de priorité

La propriété CSS font-size spécifie la taille de police

La propriété CSS font-weight définit la graisse de la police.

La propriété CSS text-decoration est utilisée pour définir la décoration du texte (formatage), telle que souligné, clignotant, barré, etc.

La propriété CSS text-transform définit comment gérer la casse du texte d'un élément. Il est ainsi possible d'afficher un texte tout en lettres capitales, tout en minuscules ou tous les mots avec leur première lettre capitale.

La propriété CSS text-align est utilisée pour décrire l'alignement des lignes de texte. text-align ne contrôle pas l'alignement des éléments, uniquement leur contenu en ligne.

La propriété CSS text-indent définie la quantité d'espace horizontal qui doit être laissée avant le début de la première ligne d'un élément contenant du texte. L'espacement horizontal se fait en accord avec la bord gauche (ou droit pour les dispositions de droite à gauche) de l'élément contenant le texte.
